Created For:

This is a personal project.

Purpose:

I created this page for two main reasons. The first, was so that I could showcase some of my previous design work, and show a live example of some things I can do. Two, so that I could experiment with some new features and code that I have not used before. I wanted it to be a place to take risks and try new things creatively.

Tools Used:

I used photoshop to mock up my original idea for the landing page, and then coded the site using HTML, CSS, and some scripts that I found across the web and modified, which used javascript.

Process:

I initially started the project by sketching out some ideas for the page designs, taking time to decide on what pages I wanted to incorporate and what information I wanted each page to include. After the sketching process I mocked up the landing page in photoshop so that I could begin creating the graphics for the hover effects. This was my first time implementing a hover effect that changed the background image of the page, so I had to do some research on how to accomplish this. I then started working on the design page, and spent some time researching different gallry scripts that would behave the way I wanted. I have never tried coding a script like this on my own, but I am interested in learning more, but starting on something much more simple than what is currently implemented on my site. The rest of the code was straight forward and didn't take long to knock out. I am still in the process of designing and coding the code page, as well as the about section and resume.

Cupcake Petri Net

This project was created in the fall of 2016 for my Modeling & Simulation class. The goal of this project was to create an interactive model of the process of baking cupcakes.

Created For:

This project was created for Paul Fishwick's Modeling & Simulation class in the fall of 2016. This was my capstone project for the course.

Purpose:

The purpose of this project was to build a model-based simulation of a scenario of my choosing.

Tools Used:

I created most of the graphics used in the project with Adobe Illustrator. I used visio to create my petri nets. The final build was built using JavaScript and HTML5 canvas.

Process:

I began this project by first compiling a list of all of the steps necessary to make cupcakes from beginning to end. I used these steps to create my first petri net. I then began creating all of the necessary illustrations. At this stage I decided to narrow down the focus of my petri net, so I went back to my diagram and removed some branches of the petri net and simplified some steps. This allowed me to focus more on the project given the short timeline to complete it. After I finished the graphics I began laying the images out using HTML5 canvas. This was my first time working with this technology. From there I added buttons for each transition and wrote the code to simulate the moving tokens using JavaScript.
